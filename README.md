## 📋 What It Does

- **🌾 Auto Farming**: Plants and harvests crops
- **💰 Auto Selling**: Sells crops in market
- **👥 Multi-Account**: Manages multiple accounts
- **🔄 Account Creation**: Creates accounts automatically
- **📊 State Management**: Tracks farm states

## 🖥️ Requirements

- **Windows 10/11**
- **MEmu Emulator**
- **Decent brain**

## 🔧 Installation

1. Download and extract
2. Run `HaydayBot.exe`

## 📱 MEmu Setup

- Resolution: 640x480
- DPI: 120
- Engine: DirectX
- Root: Enabled

## 🚀 Quick Start

1. Setup MEmu with correct resolution
2. Run `HaydayBot.exe`
3. Connect to device in Connection tab
4. Configure accounts in Account Manager tab
5. Configure farm in Farm Config tab
6. Start bot in Bot Control tab

## 🔧 Troubleshooting

**No devices found:** Restart MEmu and bot

**Wrong clicks:** Check resolution is exactly 640x480 @ 120 DPI

**App won't start:** Run as Administrator, check Windows Defender

## ❓ FAQ

**Q: Mobile support?**
A: No, MEmu only.

**Q: Multiple accounts?**
A: Yes, multiple accounts per MEmu. Multiple MEmu needs separate bots.

**Q: Adjust speed?**
A: Yes, in Farm Config tab.
